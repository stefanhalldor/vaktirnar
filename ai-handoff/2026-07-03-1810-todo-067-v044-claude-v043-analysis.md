## Blocker 1 — googleMaps.client.ts type-check villa

**Villa:** `new Loader()` er deprecated í `@googlemaps/js-api-loader` v2. TS-check brotið.

**Lagfæring (einföld):** Skipta um `new Loader()` fyrir `{ setOptions, importLibrary }` functional exports.

```ts
// Núverandi (rangt):
import { Loader } from '@googlemaps/js-api-loader'
const loader = new Loader({ ... })
loader.setOptions({ key, language: 'is', region: 'IS' })
return loader.importLibrary('places')

// Rétt (v2 API):
import { setOptions, importLibrary } from '@googlemaps/js-api-loader'
setOptions({ key, language: 'is', region: 'IS' })
return importLibrary('places')
```

Ekkert hugsunarfar — bara ein skrá, tvær línur.

## Blocker 2 — Route geocoding án notendastaðfestingar

**Vandinn:** Ef staðurinn er ekki í curated `places.ts`, kallar route handler á `geocodePlace()` og tekur `candidates[0]` beint — notandinn sér ekki hvað Google velur.

**Dæmi um vandann:** "frá Suðurgötu" — Google gæti valið Suðurgata í Reykjavík, Suðurgata á Akureyri, eða eitthvað annað. Route weather keyrist á röngu svæði án viðvörunar.

### Valkostur A — Curated-only tímabundið

Route weather virkar aðeins þegar bæði endapunktar eru í `places.ts`. Ef einn eða báðir eru ekki curated → `unknown_place` villa.

**Kostir:**
- Einföld lagfæring, ekkert nýtt UI
- Öruggt — ekkert Google guess tekið án staðfestingar
- Hægt að ship strax eftir lagfæringu

**Gallar:**
- "frá Reykjavík að Akureyri" virkar (báðir curated)
- "frá Suðurgötu að Keflavík" virkar ekki (Keflavík ekki curated enn)
- Takmarkaður fjöldi leið þar til `places.ts` stækkar

**Þegar er nóg:** Curated listi nær yfir helstu uppruna/áfangastaði (Reykjavík, Selfoss, Akureyri, Ísafjörður, Akranes, Húsavík o.fl. — 20+ staðir nú þegar). Þetta nær yfir flestar algengar spurningar.

**Kostir:**

**Gallar:**

- Þarfnast sér prófunar
