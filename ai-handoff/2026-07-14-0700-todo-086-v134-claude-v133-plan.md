### 5. Status-colored Veðurstofan map markers (v133 HIGH)

#### FerdalagidClient.tsx

Change `activeVedurstofanStationPoints` to carry status data per station:

```ts
type VedurstofanMapPoint = {
  lat: number
  lon: number
  stationId: string
  stationName: string
  status: WindDisplayStatus | 'no_data'
  windMs: number | null
  forecastTimeIso: string | null
  etaIso: string | null
}
```

Build ETA-aware status for each point in the map (same ETA logic as worstVedurstofanData loop,
but for all stations):

```ts
const activeVedurstofanStationPoints: VedurstofanMapPoint[] =
  (showVedurstofan && vedurstofanLayer)
    ? vedurstofanLayer.points
        .filter(p => p.lat !== null && p.lon !== null)
        .map(p => {
          let windMs: number | null = null
          let forecastTimeIso: string | null = null
          let etaIso: string | null = null
          if (p.routeFraction !== null && activeOutboundCandidate) {
            const depMs = new Date(activeOutboundCandidate.departureIso).getTime()
            const durMs = new Date(activeOutboundCandidate.arrivalIso).getTime() - depMs
            const etaMs = depMs + p.routeFraction * durMs
            etaIso = new Date(etaMs).toISOString()
            const nearestRow = p.forecastRows.reduce(...)
            windMs = nearestRow?.windSpeedMs ?? null
            forecastTimeIso = nearestRow?.forecastTimeIso ?? null
          } else {
            windMs = p.forecastRows.reduce((m, r) => Math.max(m, r.windSpeedMs ?? 0), 0) || null
          }
          const status: WindDisplayStatus | 'no_data' =
            windMs === null ? 'no_data'
            : windMs >= effectiveThresholds.redWindMs ? 'rautt'
            : windMs >= effectiveThresholds.cautionWindMs ? 'gult'
            : 'graent'
          return { lat: p.lat!, lon: p.lon!, stationId: p.stationId, stationName: p.stationName,
                   status, windMs, forecastTimeIso, etaIso }
        })
    : []
```

Pass updated prop to TravelAuditMap: `vedurstofanStationPoints={activeVedurstofanStationPoints}`
