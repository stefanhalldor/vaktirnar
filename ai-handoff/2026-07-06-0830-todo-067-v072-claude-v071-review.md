### P1: Cross-leg selection í buildHighlightedIssue -- STAÐFEST, ALVARLEGT

Kóðinn á línu 216 í `travel.ts`:

```ts
const { cand: worst, leg } = entries.reduce((a, b) => order[b.cand.status] >= order[a.cand.status] ? b : a)
```

`>=` þýðir: þegar status er jafnt, skilar alltaf `b`. Þar sem `returnWorst` er alltaf ýtt á listann á eftir `outboundWorst` (línur 211-212), þá er `b` alltaf return-leg þegar status er jafnt. Þetta er raunveruleg villa.

Dæmi: báðar leiðir gult, outbound vegna 16 m/s vinds, return vegna 0.5 mm/klst úrkomu -- return er valinn þrátt fyrir að outbound sé alvarlegra.

Leiðrétting:

```ts
const { cand: worst, leg } = entries.reduce((a, b) => {
  if (order[b.cand.status] > order[a.cand.status]) return b
  if (order[b.cand.status] < order[a.cand.status]) return a
  const bSev = candidateSeverity(b.cand)
  const aSev = candidateSeverity(a.cand)
  if (bSev > aSev) return b
  if (bSev < aSev) return a
  return a  // full tie: outbound first (a = outbound, appended first)
})
```

`candidateSeverity` fallið er þegar til í kóðanum frá v070.

Prófun:
- Outbound `gult` 16 m/s vindur, return `gult` 0.5 mm/h úrkoma -- assert outbound selected.
- Outbound `gult` 0.2 mm/h, return `rautt` 22 m/s -- assert return selected (rautt > gult).

### P2: Static Maps URL encoding -- STAÐFEST, RÉTTANLEGT

Núverandi `buildAuditMapUrl` notar raw string concatenation með `|` og `:` í path/markers.

Með `URLSearchParams`:
```ts
const params = new URLSearchParams({ size: '600x300', scale: '2', key: browserKey })
params.append('path', `color:0x4A90E2B0|weight:3|${pathCoords}`)
if (highlighted) params.append('markers', `color:red|label:V|${...}`)
if (dest) params.append('markers', `color:green|label:A|${...}`)
return `https://maps.googleapis.com/maps/api/staticmap?${params.toString()}`
```

Athugasemd: Google Static Maps API þiggur bæði encoded (`%7C`) og un-encoded (`|`) separatora í path og marker values. URLSearchParams encode-ar `|` sem `%7C` -- þetta er gilt og þiggst af Google. Þetta er hygiene fix, ekki mögulegt breaking change.

Prófun: unit test sem athugar að URL inniheldur `path`, `markers` (bæði), `key`, `size`.
