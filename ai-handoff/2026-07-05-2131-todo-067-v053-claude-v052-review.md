### Major 2: `trailerKind='none'` → alltaf graent

**Þetta er raunveruleg villa.**

`evalDrivingLeg` skilar alltaf `graent` þegar enginn eftirvagn er valinn — jafnvel í 25 m/s storm.
Notandi sem ferðast án eftirvagns fær misleading "Gott" niðurstöðu.

```ts
// lib/weather/travel.ts
function evalDrivingLeg(..., trailerKind: 'none' | TrailerKind) {
  if (trailerKind === 'none') return { stada: 'graent' }  // ← alltaf graent
  ...
}
```

**Lagfæring:** Bæta við einföldum general-driving þröskuldum þegar `trailerKind === 'none'`.
Til dæmis:
- gult ef vindur >= 15 m/s eða úrkoma > 0.1 mm/klst
- rautt ef vindur >= 20 m/s (stormur) eða hviður >= 28 m/s

Þessir þröskuldar þurfa að fara í `WEATHER_THRESHOLDS.driving` í `thresholds.ts` þannig að þeir
séu auðveldlega breytanlegir og prófanlegir.

**Blocker fyrir commit: Já.**

### Major 3: Stay window notar alla route points, ekki bara destination

**Þetta er raunveruleg villa sem gefur villandi niðurstöður.**

```ts
// lib/weather/travel.ts
stayConditions = worstConditions(pointForecasts, arrivalAt, returnDepartureAt)
// pointForecasts = forecast frá ÖLLUM route punktum, ekki bara destination
```

Ef vindur er 14 m/s einhvers staðar á Holtavörðuheiði á meðan notandinn er í tjaldi á Mývatni,
litast "Dvöl á Mývatni" gult jafnvel þótt Mývatn sé í lagi. Þetta er rangt.

**Lagfæring:** Í `/travel/route.ts`, sækja forecast sérstaklega fyrir destination og senda inn
sem sérstakar `destinationForecast` til `checkTravelWeather`. `travel.ts` notar þær einvörðungu
fyrir stay window.

**Blocker fyrir commit: Já.**

### Major 4: Sampling gæti misst destination punkt

**Tengd villa við Major 3 — leysast bæði saman.**

```ts
// app/api/teskeid/weather/travel/route.ts
weatherPoints.push(allPts[allPts.length - 1])  // append last
weatherPoints.splice(MAX_WEATHER_POINTS)         // ← getur fjarlægt hann aftur
```

Ef route hefur >15 punkta mun append + splice fjarlægja destination punktinn. Þetta þýðir
destination forecast er kannski aldrei sótt.

**Lagfæring:** Þar sem Major 3 fix mun sækja destination forecast sérstaklega (óháð route
sampling), leysist þetta samhliða. Route sampling nær þá yfir 0-14 punkta á leiðinni; destination
fer alltaf inn sérstaklega.

**Blocker fyrir commit: Já (leysist með Major 3 fix).**
