## Svið fyrir Codex station list verkefni

Stebbi spyr hvort Codex fái leyfi til að fylla upp stöðvalistann sem sér verkefni.

**Já, þetta er rétt hugmynd.** Codex er í betri stöðu en Claude Code til þess þar sem:
- Codex getur gert live web probes á `vedur.is/vedur/stodvar/?s={slug}` og lesið station pages
- Þetta er content/data verk, ekki kóðalogic
- Framkvæmdin er einföld: lesa coordinates af station pages, negera longitude, bæta við listann

### Hvað Codex þarf að gera

1. Finna station IDs og WGS84 coordinates fyrir helstu vegastöðvar á leiðum 1, 41, 48, 51 og Hringveginum
2. Staðfesta coordinates með því að lesa station page (ekki giska)
3. Setja `coordinatesVerified: true` aðeins þegar bæði ID og coordinates eru staðfest
4. Breyta `VEDURSTOFAN_STATIONS` array í `lib/weather/providers/vedurstofanStations.ts`
5. Keyra `npm run test:run -- lib/__tests__/weather-vedurstofan-stations.test.ts` til að staðfesta að lon sign guard og önnur tests séu enn grín
6. Keyra `npm run type-check`

### Hvað Codex á EKKI að gera

- Engar breytingar á öðrum skrám
- Engar SQL, Supabase, UI eða route.ts breytingar
- Ekki kveikja á Phase 1C eða shadow compare
- Ekki bæta við kóðalogic — aðeins data í `VEDURSTOFAN_STATIONS` array
