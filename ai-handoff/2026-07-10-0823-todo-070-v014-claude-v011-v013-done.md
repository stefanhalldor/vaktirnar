2. Curated route er ekki marktækt hraðari en fljótlegustu base route (< 60 sek mun)

**Nýjar constants:**
```ts
const HELLISHEIDI_DUPLICATE_TOLERANCE_S = 60
```

`CURATED_RING_ROAD` (Hringurinn) er undanþegin — hún getur verið lengri og birtist alltaf.

## v013: Direction-aware Hringurinn

**Skrá:** `lib/weather/google.server.ts`

### Vandamálið sem var lagað

Gamla reglan `long-trip-ring-road` notaði fast via-sequence `[Hellisheiði, suður, austur, norðaustur]` sem olli að `Reykjavík → Höfn` fór framhjá Höfn til Egilsstaðir og svo til baka — einhver slaufa.

### Lausn: tvær direction-aware reglur

**Fjarlægt:** `long-trip-ring-road` (ein broad ICELAND_BOUNDS regla)
**Fjarlægt:** `ICELAND_BOUNDS` constant

**Bætt við bounds:**
```ts
// Norður/norðaustur Island: Akureyri (65.68), Mývatn (65.60), Húsavík
const NORTH_ICELAND_RING_ROAD_BOUNDS = { minLat: 65.40, maxLat: 66.7, minLon: -22.0, maxLon: -14.0 }

// Suðausturströnd: Höfn (64.25), Djúpivogur, Breiðdalsvík
const SOUTHEAST_COAST_RING_ROAD_BOUNDS = { minLat: 63.5, maxLat: 65.0, minLon: -15.9, maxLon: -13.0 }
```

**Bætt við via-punkt:**
```ts
const RING_ROAD_NORTH_VIA = { lat: 65.540, lon: -19.520 } // Route 1, Varmahlíð — pending verification
```

**Tvær nýjar reglur:**

```ts
// Regla A: gagnvísar (suður-austur-norður) fyrir norðlægar áfangastaðir
{
  id: 'long-trip-ring-road-south-east-north',
  origin: { bounds: [CAPITAL_AREA_BOUNDS] },
  destination: { bounds: [NORTH_ICELAND_RING_ROAD_BOUNDS] },  // Akureyri, Mývatn
  minFastestRouteDistanceM: 350_000,
  vias: [HELLISHEIDI_VIA, RING_ROAD_SOUTH_VIA, RING_ROAD_EAST_VIA, RING_ROAD_NORTHEAST_VIA],
  labels: ['CURATED_RING_ROAD'],
}

// Regla B: réttsælis (norður-austur-suður) fyrir suðausturströnd
{
  id: 'long-trip-ring-road-north-east-south',
  origin: { bounds: [CAPITAL_AREA_BOUNDS] },
  destination: { bounds: [SOUTHEAST_COAST_RING_ROAD_BOUNDS] },  // Höfn, Djúpivogur
  minFastestRouteDistanceM: 350_000,
  vias: [RING_ROAD_NORTH_VIA, RING_ROAD_NORTHEAST_VIA],  // 2 vias, ekki framhjá Höfn
  labels: ['CURATED_RING_ROAD'],
}
```

**Egilsstaðir (lat 65.268) er í bili utan beggja reglna** (of lágt fyrir NORTH minLat 65.40, of hátt fyrir SOUTHEAST maxLat 65.0). Hringurinn birtist ekki þar í bili — öruggara en brotnar leiðir.

2. Ástarlegar leiðir (t.d. `Garðabær → Egilsstaðir`)
   - `Um Hellisheiði` á enn að birtast þar sem Google skilur gjarnan eftir þessa leið.

### v013: Direction-aware Hringurinn

1. **`Reykjavík → Akureyri`** (lykilpróf)
   - Hringurinn birtist, fer suður/austur/norður um Þjóðveg 1.
   - Kortið: Hellisheiði → suðurströnd → austurland → norður → Akureyri.
   - Engar slaufur inn í bæjarstrætum.

2. **`Reykjavík → Höfn`** (lykilpróf — þetta var brotin leið)
   - Hringurinn birtist, fer norður/austur/suður — **ekki** framhjá Höfn og til baka.
   - Kortið: Reykjavík → norður (Varmahlíð-svæðið) → austurland → suður → Höfn.
   - Ef via-punktar eru rangar og Google skilar enn vitlægri leið → leiðréttum hnit.

3. `Reykjavík → Egilsstaðir`
   - Hringurinn birtist **ekki** (utan beggja destination bounds).
   - `Um Hellisheiði` birtist enn.

4. `Garðabær → Mývatn` (ef leið er > 350 km)
   - Hringurinn birtist og fer suður/austur/norður (Regla A — Mývatn er í NORTH bounds).

5. `Reykjavík → Þorlákshöfn`
   - Engin Hringurinn (< 350 km, og utan destination bounds).
   - Engin `Um Þrengslaveg` (rule fjarlægð).

6. Veldu Hringurinn á hverri leið og haltu áfram → engin `selected_route_unavailable`.
