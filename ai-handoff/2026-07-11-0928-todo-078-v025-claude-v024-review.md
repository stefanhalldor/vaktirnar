## P1 — fail-open ferdalagid (rétt greining)

Codex er rétt. Núverandi kóði:

```ts
if (process.env.WEATHER_TRIP_FLAG !== 'true') return true  // ← fail-open!
```

Þetta þýðir: allir vedrid-notendur sjá affordancena þegar flagið er ekki sett.
Þetta er andstætt prerelease-markmiðinu.

Lagfæring er ein lína:

```ts
if (process.env.WEATHER_TRIP_FLAG !== 'true') return false  // fail-closed
```

`.env.example` comment þarf líka að uppfærast.
