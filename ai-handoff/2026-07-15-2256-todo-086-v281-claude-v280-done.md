**`toPublicFirstName(displayName)`** — nýr einkafallshjálpar:
```ts
function toPublicFirstName(displayName: string | null): string | null {
  if (!displayName) return null
  const first = displayName.trim().split(/\s+/)[0]
  return first || null
}
```

### `app/auth-mvp/vedrid/elta-vedrid/VedurstofanStationExplorerClient.tsx`

`WeatherPulseSummary` endurskrifað með tveimur sjálfstæðum hlutum:

**Public preview:**
- Sækir frá `/api/teskeid/weather/vedurpuls/stations/${stationId}/preview` á mount og í 30s interval
- Sýnir `pulseLoading` meðan `!previewLoaded`
- Sýnir `ChatPreviewList` þegar búið (eða pulseEmpty ef tómt)
- Polling er algjörlega óháð auth

**Lazy compose (auth-gated):**
- "Skrifa umferðarfrétt" hnappur alltaf sýnilegur undir preview
- Smella opnar composer; smella aftur lokar honum
- Fyrsta kall á POST /thread fer út þegar compose opnast
  - 401/403/503 → `accessDenied=true` → sýnir `pulseAccessDenied`
  - Villa → lokar compose, hnappur heldur áfram að virka
  - OK → `threadId` sett, inline input birtist
- Input: `autoFocus`, `maxLength={1000}`, Enter sendir
- Eftir sendingu: compose lokast, preview refreshast með nýjustu skilaboðum
- `sendError` setur body til baka í input

"Opna púlsinn" tengill er áfram til staðar til hliðar við "Skrifa umferðarfrétt".
